# Content Formula

## Duration
20 to 40 seconds

## Fixed Structure
1. Hook: 0-2s
2. Problem: 2-6s
3. Escalation: 6-15s
4. Turning Point: 15-25s
5. Emotional Ending: 25-40s

## Rules
- One idea only.
- One emotional arc only.
- Every 2-3 seconds must change visually or emotionally.
- First frame must create curiosity.
- Ending must loop back to the start.
- No filler sentences.
