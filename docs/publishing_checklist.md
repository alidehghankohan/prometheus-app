# Publishing Checklist

## Before Export
- Hook is strong.
- Script is 20-40 seconds.
- Visual beats are clear.
- Voiceover is ready.
- Loop ending exists.

## Before Upload
- Title is short.
- Caption includes comment CTA.
- Hashtags are relevant.
- Cover frame is readable.
- No dead space in the edit.

## After Upload
- Check retention.
- Check comments.
- Check replays.
- Save winning structure.
- Improve next episode.
