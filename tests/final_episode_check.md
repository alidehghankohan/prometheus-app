# Final Episode Check

## 5 Checks
- Hook works in first 2 seconds.
- Every 2-3 seconds something changes.
- Ending loops back to the start.
- On-screen text is readable on mobile.
- CTA is short and comment-friendly.

## Pass Rule
- If all 5 pass, publish.
- If 1 or more fail, revise before upload.

## Revision Targets
- Hook
- Pace
- Ending
- Caption
- CTA
