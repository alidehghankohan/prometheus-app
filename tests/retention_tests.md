# Retention Tests

## Retention Pass Rules
- First 2 seconds must create curiosity.
- No dead air.
- New visual change every 2-3 seconds.
- One clear emotional arc.
- Ending must loop.

## Sample Pass
- Message appears on the wall.
- Neo freezes.
- Byte reacts.
- The screen glitches again.
- End connects to the beginning.

## Fail Examples
- Long intro.
- Slow explanation.
- Repeated visuals.
- No twist.
- Flat ending.
