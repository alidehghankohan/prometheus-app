# Post Publish Review

## After 24 Hours
- Check views.
- Check average view duration.
- Check retention at 2 seconds.
- Check comments.
- Check replays.

## Decision
- If hook is weak, rewrite hook.
- If retention drops, tighten pacing.
- If comments are low, improve CTA.
- If title underperforms, test a new title.
- If all metrics are good, reuse the structure.
