# Hook Tests

## Hook Pass Rules
- Hook must work in 0-2 seconds.
- Hook must create curiosity or tension.
- Hook must be short and clear.
- Hook must fit the universe.
- Hook must not start with greeting or intro.

## Sample Hooks
- The glitch appeared again.
- Neo saw his name on the wall.
- Byte found a message from tomorrow.

## Fail Hooks
- Hi guys, welcome back.
- Today I will show you...
- Let me explain something...
